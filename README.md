## Installation 

Make sure you have `julia` installed. I recommend [juliaup](https://github.com/JuliaLang/juliaup). 

```
git clone https://github.com/hollisakins/Lazy.jl.git
cd Lazy.jl
bash install.sh
```
This will install the `lazy` executable at `~/.julia/bin/`.
